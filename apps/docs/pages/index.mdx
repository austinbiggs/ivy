# Ivy UI

Welcome to the **Ivy UI** documentation. This library provides a comprehensive set of components designed to build responsive and modern web applications.

## Button

```jsx
<Button variant="primary">Click Me</Button>
```

### Props

- `variant`: Defines the button style. Options: `primary`, `secondary`, `danger`.
- `onClick`: Function to handle the click event.

## Card

```jsx
<Card title="Card Title" subtitle="Card Subtitle">
  <Typography>This is a card component from Ivy UI.</Typography>
</Card>
```

### Props

- `title`: The title of the card.
- `subtitle`: A subtitle for additional context.
- `children`: The content inside the card.

## Typography

```jsx
<Typography variant="h1">Heading 1</Typography>
```

### Variants

- `h1` - Heading 1
- `h2` - Heading 2
- `h3` - Heading 3
- `body` - Body text
